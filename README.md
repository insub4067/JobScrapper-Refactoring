# Remot Job Srapper

> **Explanation** 
<br>
1. Scrap Jobs from 3 different sites which is 👇👇
Stack Overflow, WeWorkRemotely, RemoteOK
2. Show as results as table
3. You may download results as CSV file


> **Used Skills**
<br>
Backend : Flask
Scrap : BeautifulSoup4
Async : Asyncio
request : aiohttp
css : Tailwind CSS
